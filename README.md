## Инструкции

### Инструкция по сборке
За сборку приложения отвечает файл **BUILD.sh**

### Инструкция по запуску
За запуск приложения отвечает файл **RUN.sh**

### Инструкция по тестированию
За тестирование приложения отвечает файл **TEST.sh**
